# ChatGPT Widget

This package contains the React UI that ChatGPT renders for Moodle tools.

The source uses:

- Vite
- React
- Tailwind CSS v4
- shadcn/ui components generated by the shadcn CLI
- PDF.js for the embedded PDF viewer

Build and copy the single-file HTML artifact into the Go MCP server:

```bash
bun run build:chatgpt-widget
```

The Go server embeds `pkg/chatgptapp/widget_dist.html` via `go:embed`.

When changing UI components, use the shadcn CLI from this directory:

```bash
bunx --bun shadcn@latest info
bunx --bun shadcn@latest add button
```
